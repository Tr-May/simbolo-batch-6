CSS Selector : ID and Class Selector

ID Selector

An ID selector in CSS is used to target a single unique element on a web page. The ID attribute of an HTML element provides a unique identifier for that element within the document. ID selectors are denoted by a hash (#) followed by the ID name.

Class Selector

A Class selector in CSS is used to target one or more elements that share the same class attribute. Classes allow you to apply the same styles to multiple elements across the document. Class selectors are denoted by a period (.) followed by the class name.
